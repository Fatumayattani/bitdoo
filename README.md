# 🌍 Bitdoo

**Empowering global communities with decentralized Bitcoin microloans. Built on ICP.**

![Bitdoo Banner](./public/bitdoo.png)

---

## 💡 Inspiration

In many regions of the world, especially in developing countries, millions of people remain unbanked or underbanked. Traditional microfinance systems often involve high interest rates, middlemen, and little transparency. 

We wanted to build a trustless, transparent system that lets people **borrow and lend Bitcoin directly** — no bank required. With **Chain Fusion on the Internet Computer (ICP)**, we can now interact with Bitcoin natively in smart contracts, unlocking programmable Bitcoin use cases.

---

## 🚀 What it Does

**Bitdoo** is a decentralized microloan platform where:
- Borrowers request Bitcoin loans with defined terms.
- Lenders fund loans trustlessly, with funds held in escrow via smart contracts.
- Smart contracts track repayments and enforce deadlines.
- A **reputation system** incentivizes honest behavior and builds borrower trust.
- All transactions are transparent and verifiable on-chain.

Key Features:
- 📄 Loan Requests with customizable terms
- 🔐 Smart contract-based Bitcoin escrow
- 📆 Automated repayment tracking
- ⭐ On-chain reputation score
- 📊 Lender dashboard and performance tracking

---

## 🛠️ How We Built It

- **Frontend**: [Next.js 13](https://nextjs.org/) with TailwindCSS, hosted on [Netlify](https://www.netlify.com/)
- **Backend**: Smart contracts written in [Motoko](https://internetcomputer.org/docs/current/motoko) and deployed to ICP canisters
- **Bitcoin Integration**: Leveraged [ICP Chain Fusion](https://internetcomputer.org/bitcoin-integration) to interact directly with the Bitcoin network
- **State Management**: React Hook Form + Zod
- **Charts & Visuals**: Recharts, Radix UI components
- **Wallets**: Bitcoin testnet wallets (for development)

---

## 🎯 Challenges We Ran Into

- Setting up and testing **Bitcoin Chain Fusion** correctly
- Designing secure, fair **repayment logic** within smart contracts
- Managing **identity and reputation** in a decentralized, privacy-respecting way
- Dealing with **edge cases** like partial repayments, delayed funding, etc.

## 🏆 Accomplishments We’re Proud Of

- Fully functional loan request & funding workflow in the frontend
- Secure Bitcoin escrow system powered by ICP
- On-chain trust score that updates based on repayments
- Clean, mobile-friendly UI for accessibility in low-infra environments
